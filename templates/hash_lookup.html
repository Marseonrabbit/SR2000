What is a File Hash?

A file hash is a unique fingerprint for a file, generated by running the file's data through a mathematical algorithm. Hashes are used to:

  • Verify file integrity
  • Identify malware and viruses
  • Track known good/bad files
  • Check if files have been modified
  • Compare files without sharing content
  • Identify files across different systems
